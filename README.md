## 🎯 Learning Path
1. **Image Processing**: Master OpenCV for data augmentation and cleaning.
2. **Real-time Detection**: Learn to optimize YOLO for edge devices.
3. **Zero-shot Vision**: Use CLIP and SAM for generalized vision tasks.
4. **Deployment**: Optimize CV models with TensorRT.
