The breakthrough papers that paved the way for modern Computer Vision.

## The Foundations
1.  **ImageNet Classification with Deep Convolutional Neural Networks (2012)**: "AlexNet" - The paper that started the Deep Learning revolution. [Read Paper](https://proceedings.neurips.cc/paper/2012/file/c399862d3b9d6b76c8436e924a68c45b-Paper.pdf)
2.  **Deep Residual Learning for Image Recognition (2015)**: Introducing **ResNet** and skip connections. [Read Paper](https://arxiv.org/abs/1512.03385)

## Detection & Segmentation
3.  **You Only Look Once: Unified, Real-Time Object Detection (2015)**: The birth of **YOLO**. [Read Paper](https://arxiv.org/abs/1506.02640)
4.  **Faster R-CNN: Towards Real-Time Object Detection with Region Proposal Networks (2015)**: [Read Paper](https://arxiv.org/abs/1506.01497)
5.  **U-Net: Convolutional Networks for Biomedical Image Segmentation (2015)**: [Read Paper](https://arxiv.org/abs/1505.04597)

## The Modern Era (Foundation Models)
6.  **An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale (2020)**: **ViT** - Proving Transformers can work for Vision. [Read Paper](https://arxiv.org/abs/2010.11929)
7.  **Segment Anything (SAM, 2023)**: Meta's foundation model for segmentation. [Read Paper](https://arxiv.org/abs/2304.02643)

---

> [!TIP]
> **Must Read**: Start with **YOLO (2015)**. It explains the "Detection as Regression" philosophy very clearly and changed how we think about real-time vision.
